Playlist Generator (JS, Node, Express, mySQL, HTML, CSS)
iPlaylist: This was my second group project while attending the UPenn coding bootcamp. I worked on a team of three and we built a music playlist generator application. This program relied solely on deezers API for getting a list of songs. There were two overall themes for the project. We wanted to give the users the ability to create their own playlist but also contribute to what we called the "community playlist". This would be a playlist that is contanstly updated based on a group of users via a voting mechanism (likes/dislikes). Unlike the song sparrow application, we deployed this project publicly via heroku and built an entire backend using node.js, express, and sequelize. A cool feature that we implemented using the passport library was a fully functional login system that encrypts and decrypts users passwords to ensure security.

Music Song and Event Info Search (JS, HTML, CSS)
Song Sparrow: This was my first group project during my coding bootcamp. I worked on a team of three to build a web application where a user can search for music related to an artist and personalize their own music page. We used multiple API's such as genius and seatgeek to get up to date music and event information. In addition to search features we created a chat feature for users to communicate with one another. This includes only basic validation, for example limiting the user from uplaoding unwanted scripts. We did not use a traditional relationship based database, instead we stored our user data in google clouds firebase. The project is deployed live via github pages.
